SCM Redesign

Overview

As a practice project, this redesign allowed me to refine my research, wireframing, and Figma skills while exploring practical product design solutions. I focused on the Steam Community Market, addressing usability challenges and pain points for CS2 items, the platform’s largest category.

Research

• User Surveys: Collected input from traders and investors in CS2, in order to uncover frustrations with the current marketplace aswell as desired features.

• Competitve analysis: I conducted a analysis of the Steam Community Market alonside its third part market competitors in order to see what features it lacked, aswell for layout inspiration.

Key Insights

The Key Insights that guided my design decisions were as follows:

• Navigation: Wanted to be able to click on a button on the page and not need to navigate menus to swtich pages

• Item Browsing: Desired increased filter options like price and float

• Sorting: Users wanted a clear and easy to use sorting button with better sorting critiera

• Autofill: Users wanted to be able to autofill the name of items rather then having to type them out fully

Process in Figma

I began by manually translating the Steam background and key interface elements into my wireframe. Next, I designed a mid-fidelity menu.

Afterwards, I added high level deatail to the midfi wireframe in order to create a hifi wireframe that can later be used in a prototype.

Reflection

This practice project gave me valuble experience in problem solving, user reaserach, applying design heuristics, and refining my Figma workflow. It also inspired my Chrome Extension CS Search, giving me insights that informed my design.
